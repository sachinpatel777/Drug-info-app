# Drug Information Portal

A React-based web application to view and filter drug information.

## Features
- **Data Display**: Shows drug information in a professional table using Material UI.
- **Sorting**: Automatically sorts data by descending launch date.
- **Filtering**: Filter drugs by company using a dropdown or by clicking the company name in the table.
- **Unit Testing**: Includes a test suite to verify filtering logic.
- **Locale Formatting**: Displays dates according to the user's local system settings.

## Technologies Used
- React.js
- Material UI (MUI)
- Jest & React Testing Library (for unit tests)
- JSON (Local data source)

## How to Run Locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to view the app in your browser.
4. Run `npm test` to execute unit tests.
